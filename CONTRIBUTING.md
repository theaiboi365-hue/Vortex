# Contributing

Thanks for helping improve Vortex.

## Good First Contributions

- Improve Windows setup reliability.
- Add screenshots and short demo GIFs.
- Add provider adapters.
- Improve Slack and Telegram command handling.
- Add tests for provider routing.
- Improve docs for non-technical users.

## Development

```powershell
npm.cmd install
npm.cmd run check
npm.cmd start
```

Open:

```text
http://127.0.0.1:8787
```

## Pull Request Checklist

- Run `npm.cmd run check`.
- Do not commit `.env`.
- Keep setup steps clear for Windows users.
- Keep provider routing backwards-compatible.
- Avoid promotional claims in docs unless they are clearly framed as launch copy.
