# Wikipedia-Safe Plan

## Important Boundary

Do not add Vortex links to unrelated Wikipedia pages, including pages for other projects, unless independent reliable sources explicitly discuss Vortex in that context.

Wikipedia is not a launch channel. Promotional editing can be reverted quickly and can damage the project reputation.

## What Is Safe

- Keep a neutral draft ready.
- Track independent reliable coverage.
- Use edit requests instead of direct editing when there is a conflict of interest.
- Only propose additions to related pages if the statement is backed by independent reliable sources.

## What Is Not Safe

- Adding the GitHub link to unrelated pages.
- Creating a page before there is notability.
- Adding comparisons to OpenClaw, Hermes, or other pages without independent sources.
- Using the repo README, personal posts, or social posts as the only sources.

## Pages To Watch Later

Only after reliable third-party coverage exists:

- Pages about software agents
- Pages about chatbots
- Pages about Slack bots or Telegram bots if relevant
- Pages about AI coding assistants if reliable sources explicitly connect the project
