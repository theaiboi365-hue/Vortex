# Wikipedia Source Tracker

Wikipedia needs independent reliable sources. GitHub, the project README, social posts, and self-published blogs are useful for users but weak for notability.

| Source | Independent? | Reliable? | Significant coverage? | Notes |
| --- | --- | --- | --- | --- |
| GitHub repo | No | Primary source | No | Useful for facts only, not notability |
| Project README | No | Primary source | No | Do not use for notability |
| Product Hunt launch | Partly | Usually weak | Usually no | Useful for launch presence, not strong notability |
| News article | Yes | Depends | Depends | Best candidate |
| Independent technical review | Yes | Depends | Depends | Good if detailed and editorial |
| Academic paper | Yes | Usually strong | Depends | Good if it discusses the project substantively |

## Minimum Goal Before Wikipedia

Collect at least 3 independent reliable sources with significant coverage.

## Candidate Outreach Targets

- Independent AI tooling newsletters
- Developer tool blogs with editorial standards
- Open-source roundups
- Local AI newsletters
- Slack/Telegram automation writeups
