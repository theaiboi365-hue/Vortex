# Launch Plan

## Core Hook

Codex from your phone in 60 seconds.

## First 24 Hours

1. Pin the GitHub repo in the browser and make sure the README shows the install command above the fold.
2. Record a short demo:
   - Run the PowerShell installer.
   - Show the setup dashboard.
   - Add Telegram token.
   - Send a Telegram message.
   - Show Codex replying.
   - Switch Telegram provider to Ollama or Anthropic.
3. Add the GIF/video to the top of the README.
4. Post to X, Reddit, LinkedIn, and Hacker News.
5. Ask early users to open issues for install friction.

## GitHub Polish Checklist

- README hero has one-command installer.
- Logo is visible.
- Badges are visible.
- Demo GIF is visible.
- `INSTALL-WINDOWS.md` exists.
- `docs/PRESS-KIT.md` exists.
- `docs/COMPARISONS.md` exists.
- Issues are enabled.
- Topics should include: `codex`, `slack-bot`, `telegram-bot`, `ai-agent`, `ollama`, `windows`, `socket-mode`, `openai`, `automation`.
