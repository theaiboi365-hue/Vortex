# Codex Social Bridge v1.0.0

Codex Social Bridge is ready for its first public release.

## Main Hook

Run Codex from Telegram and Slack with one PowerShell command.

## Install

```powershell
powershell -ExecutionPolicy Bypass -Command "irm https://raw.githubusercontent.com/theaiboi365-hue/codex-social-bridge/main/scripts/setup-windows.ps1 | iex"
```

## Highlights

- Codex as the default AI brain.
- Telegram and Slack support from one backend.
- Local dashboard for tokens and provider routing.
- Windows startup automation.
- Hidden background runner.
- Switchable AI brains per surface.
- Launch kit and Wikipedia-safe docs included.
